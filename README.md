# SynapseLogic - Simulador de Conectividad y Transmisión Neuronal
Autores: Armando Suarez, Rocco García, Gabriel Cedeño

SynapseLogic es una aplicación interactiva diseñada para modelar, visualizar y analizar la transmisión de impulsos eléctricos en una red neuronal mediante el uso de la teoría de grafos.
El núcleo del sistema destaca por cumplir con la restricción estricta de no utilizar librerías de colecciones prefabricadas de Java.

Los archivos Jar para poder correr el Graphstream Estan incluidos en el repositorio.
